
========= GBA ALIEN GAME ==========
Description:
    Alien needs help landing his ship. Goal is to make it to the landing pad without crashing the $10000000000 ship.

Controls:

ENTER        : starts game
BACKSPACE    : restarts game
LEFT ARROW   : move left
RIGHT ARROW  : move right
S KEY        : move up
